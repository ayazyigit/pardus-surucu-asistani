Pardus Sürücü Asistanı Kurulum Adımlar;
Konsola giriniz.
"cd Masaüstü" komutunu yazınız.
"git clone https://github.com/ayazyigit/pardus-surucu-asistani" komutunu yazınız.
Masaüstü'nde bulunan "pardus-surucu-asistani" klasörünü açınız.
Klasörün içindeki "pardus-paket.deb" dosyasını çalıştırınız.
Başlatma menüsünden "Pardus Sürücü Asistanı" ismini aratıp uygulamayı başlatınız.

Pardus Sürücü Asistanı Kaldırma Adımları;
Pardus Yazılım Merkezi'ne girin.
Üst taraftaki depo sayfasına girin.
Depo sayfasındayken "pardus-driver-assistant" aratmasını yapın.
Çıkan ilk deponun üstüne basın ve sağ tarafta açılan kırmızı "Kaldır" butonuna basın.
