Pardus Sürücü Asistanı Kurulum Adımlar;
1. Konsola giriniz.
2. "cd Masaüstü" komutunu yazınız.
3. "git clone https://github.com/ayazyigit/pardus-surucu-asistani" komutunu yazınız.
4. Masaüstü'nde bulunan "pardus-surucu-asistani" klasörünü açınız.
5. Klasörün içindeki "pardus-paket.deb" dosyasını çalıştırınız.
6. Başlatma menüsünden "Pardus Sürücü Asistanı" ismini aratıp uygulamayı başlatınız.

Pardus Sürücü Asistanı Kaldırma Adımları;
1. Pardus Yazılım Merkezi'ne girin.
2. Üst taraftaki depo sayfasına girin.
3. Depo sayfasındayken "pardus-driver-assistant" aratmasını yapın.
4. Çıkan ilk deponun üstüne basın ve sağ tarafta açılan kırmızı "Kaldır" butonuna basın.
